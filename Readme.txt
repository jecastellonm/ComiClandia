Instrucciones de uso
- Ejecutar script de creacion tablas Base de datos ComiCLandia  Script_DB_ComiClandia.sql
- Ejecutar scripts de creacion de procedures Script_ProcedureCreateEditCliente.sql  Script_ProcedureCreateEditProducto.sql   Script_ProcedureCreateEditDetallePedido.sql Script_ProcedureCreateEditPedido.sql
- Importar data de prueba a la base de datos ComiClandia con los archivos ComiClandiaCliente.txt y ComiClandiaProducto.txt
- Ejecutar Aplicacion Manejo de Pedidos ComiClandia
Estados de los Pedidos
1 Pendiente, 2 Alistando, 3 Entregado, 4 En Cobro, 5 Pagado 6 Cancelado
Tipo Pedido 
1 Entrega Inmediata,  2 Domicilio,  3 Retiro en TIenda
